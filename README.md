```
An Ubuntu/Debian packaging script for Apache Kafka

USAGE: ./build.sh VERSION REVISION

./build.sh 0.7.2 1

VERSION: the Kafka release version (eg: 0.7.2)
REVISION: the revision number to use when generating the .deb (eg: 1)
```
